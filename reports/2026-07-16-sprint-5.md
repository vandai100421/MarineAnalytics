**Geofence Engine (T5.2):**
- Khi `write_position()` được gọi → `check_position_against_geofences()`
- Query `ST_Contains(geofence.geom, POINT(lon lat))` tìm tất cả geofence chứa vị trí
- Nếu tàu mới vào geofence → insert alert `event_type='enter'`
- Dedup: skip nếu đã có alert 'enter' cho cùng (mmsi, geofence_id) trong 30 phút
- Error resilient: catch exception, log, không crash ingestion

**API Endpoints:**
```
POST   /api/v1/geofences          (name, type, coordinates[], description)
GET    /api/v1/geofences           → list all
GET    /api/v1/geofences/{id}      → single
DELETE /api/v1/geofences/{id}      → 204
GET    /api/v1/alerts?from=...&geofence_id=...&limit=100&offset=0
```

**Frontend:**
- GeofenceEditor: draw polygon (click map thêm points), name + type input, save/clear
- AlertPanel: danh sách alert gần đây (enter/exit, MMSI, geofence, thời gian, tọa độ), auto-refresh 30s
- Tab "Geofence" trong sidebar (Map/Dashboard/Geofence)
