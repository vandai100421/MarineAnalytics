**ADS-B Ingestion:**
- `poll_adsbexchange()`: poll REST API mỗi 10 giây, tự động skip nếu không có API key
- `adsb_writer.py`: batch insert vào `aircraft_positions` hypertable + Redis cache `air:{hex}` (TTL 10min)
- Parse: hex, lat, lon, alt_baro, gs (ground speed), track, flight, reg, type
- Lifespan: start ADS-B task cùng AIS ingestion task, cancel cả hai khi shutdown

**Aircraft API:**
```
GET /api/v1/aircraft/positions?bbox=100,10,120,20
```
Đọc từ Redis (realtime), filter theo bbox.

**Frontend:**
- AircraftLayer: ScatterplotLayer màu tím (purple), highlight vàng khi selected
- Toggle 4 mode: Vessels / Aircraft / Both / Heatmap (button bottom-left)
- AircraftInfo panel: flight callsign, hex, registration, type, altitude, ground speed, track, position, updated time
- mapStore: `selectedHex`, `mapMode` expanded với 'aircraft' và 'both'
