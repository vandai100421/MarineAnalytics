**Thay đổi kiến trúc:**
- Tạo `repositories/` module: `VesselRepository`, `PositionRepository` (CODING_STANDARDS yêu cầu repo pattern)
- Đổi `redis.keys("pos:*")` → `redis.scan_iter(match="pos:*")` (tránh block Redis)
- Tạo `core/errors.py`: RFC 7807 ProblemDetail (type, title, status, detail)
- Endpoint dùng `Depends(get_redis)` thay vì gọi trực tiếp (testable)
- Thêm filter `ship_type` và `min_sog` cho positions endpoint
- Track endpoint: downsample limit 5000 points, trả `TrackResponse` với total count

**API Endpoints:**
```
GET /api/v1/vessels/positions?bbox=100,10,120,20&ship_type=70&min_sog=5
GET /api/v1/vessels/{mmsi}
GET /api/v1/vessels/{mmsi}/track?from=2026-07-16T00:00:00Z&to=2026-07-17T00:00:00Z&limit=5000
```

**Error format (RFC 7807):**
```json
{
  "type": "about:blank",
  "title": "Not Found",
  "status": 404,
  "detail": "Vessel 999 not found"
}
```
