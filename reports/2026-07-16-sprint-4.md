**Track Playback:**
- TimelineScrubber: play/pause, reset, speed control (1x/2x/4x/8x), slider progress
- PathLayer vẽ path từ track API, animate theo playbackIndex
- mapStore thêm `playbackIndex` + `setPlaybackIndex`
- Khi chọn tàu → query track → hiển thị scrubber + path animation

**Track Downsample (T4.3):**
- Nếu total points ≤ 5000: query tất cả
- Nếu total > 5000: query tất cả rồi slice `[::step]` (step = total // limit)
- Đảm bảo response < 5000 points để frontend mượt

**Heatmap (T4.4 + T4.5):**
- HexagonLayer với colorRange 6 cấp (xanh → đỏ)
- Data từ realtime positions (lat/lon)
- Toggle Vessels/Heatmap button ở bottom-left map
- Heatmap endpoint hỗ trợ bbox + time range filter
